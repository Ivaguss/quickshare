# QuickShare — обмен между устройствами

Общая «комната» для быстрого обмена **текстом, фото и файлами** между iPhone, Android, iPad и Mac.
Работает в браузере, через интернет. Заходишь по одному коду комнаты на всех устройствах — видишь общий список,
всё синхронизируется мгновенно.

## Возможности
- 📋 Текст и ссылки — отправка, копирование в один тап, авто-подсветка ссылок
- 🖼️ Фото и файлы — загрузка, превью картинок, скачивание, копирование картинки в буфер
- 📥 Вставка из буфера (Ctrl/⌘+V) и перетаскивание файлов
- 🔗 Кнопка «поделиться» ссылкой на комнату (для быстрого открытия на другом устройстве)
- 📱 Устанавливается как приложение (PWA) на телефон и Mac
- 🌗 Тёмная и светлая тема автоматически
- ⏱️ Данные временные: авто-очистка старого, лимиты по размеру

> ⚠️ Это временный буфер обмена: содержимое хранится в памяти сервера и удаляется по таймауту
> (по умолчанию 6 часов) или при перезапуске. Не для постоянного хранения.
> Кто знает код комнаты — видит её содержимое. Используйте длинные/случайные коды.

## Запуск локально
```bash
cd quickshare
npm install
npm start
# откройте http://localhost:3000
```
Чтобы открыть с телефона в той же Wi-Fi сети — используйте локальный IP компьютера, например `http://192.168.1.50:3000`.

## Деплой на готовый хостинг (через интернет)

Нужен хостинг, который держит постоянный процесс и WebSocket (обычные «serverless» не подходят).
Самые простые бесплатные/дешёвые варианты:

### Render (рекомендую, есть бесплатный тариф)
1. Загрузите папку `quickshare` в репозиторий на GitHub.
2. На https://render.com → **New → Blueprint** → выберите репозиторий (в нём есть `render.yaml`).
3. Через пару минут получите адрес вида `https://quickshare-xxxx.onrender.com` — открывайте его на всех устройствах.
   > На бесплатном тарифе сервис «засыпает» без трафика — первое открытие после паузы занимает ~30 сек.

### Railway / Fly.io / любой VPS
- Railway: New Project → Deploy from GitHub → он сам подхватит `package.json`.
- Fly.io / VPS / Docker: используйте `Dockerfile` (`docker build -t quickshare . && docker run -p 3000:3000 quickshare`).

## Настройки (переменные окружения)
| Переменная | По умолчанию | Значение |
|---|---|---|
| `PORT` | 3000 | Порт |
| `MAX_FILE_BYTES` | 30 МБ | Макс. размер одного файла |
| `ROOM_QUOTA_BYTES` | 150 МБ | Лимит на комнату |
| `MAX_ITEMS` | 200 | Макс. элементов в комнате |
| `ITEM_TTL_MS` | 6 ч | Через сколько удалять старые элементы |
